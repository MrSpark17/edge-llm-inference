# Performance Benchmarks

Testing conducted on: **Intel i5 (Mobile) | 8GB Total RAM (Limited to 2GB via Docker)**

| Metric | Value |
| :--- | :--- |
| **Avg. Inference Latency** | 840ms |
| **p95 Latency** | 1.1s |
| **Peak Memory Usage** | 1.84 GB |
| **Model Load Time** | 2.4s |
| **Intent Accuracy (Validated)** | 94.8% |

*Note: Benchmarks represent average of 50 runs with a warm KV-cache.*
